Classes

AdminPageFramework_FieldType The base class for the users to create their custom field types.
AdminPageFramework_FieldType_checkbox A check box that lets the user enable/disable an option item.
AdminPageFramework_FieldType_color A text field with a color picker.
AdminPageFramework_FieldType_export Defines the export field type.
AdminPageFramework_FieldType_file A file uploader that lets the user upload files.
AdminPageFramework_FieldType_hidden A field with invisible input values.
AdminPageFramework_FieldType_image A text field with an image uploader.
AdminPageFramework_FieldType_import Defines the import field type.
AdminPageFramework_FieldType_inline_mixed A field that includes child fields with different field types.
AdminPageFramework_FieldType_media A text field with a media uploader lets the user set a file URL.
AdminPageFramework_FieldType_number A number field that lets the user set numbers.
AdminPageFramework_FieldType_posttype A post type field can list available post types on the site.
AdminPageFramework_FieldType_radio A set of radio buttons that lets the user pick an option.
AdminPageFramework_FieldType_section_title Defines the section_title field type.
AdminPageFramework_FieldType_select A select field lets the user select drop-down list items.
AdminPageFramework_FieldType_size A combination field of a number and a select inputs that let the user set sizes.
AdminPageFramework_FieldType_submit Defines the submit field type.
AdminPageFramework_FieldType_system Defines the system field type.
AdminPageFramework_FieldType_taxonomy A taxonomy field can list terms of specified taxonomies.
AdminPageFramework_FieldType_text A text field that lets the user set short text values.
AdminPageFramework_FieldType_textarea A text area field that lets the user set text values with multiple lines.
